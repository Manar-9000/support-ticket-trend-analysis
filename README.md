# Support-ticket-trend-analysis

## Overview
Analysis of 35 sample IT support tickets to identify recurring 
issue patterns and recommend documentation/training improvements.

## Method
- Filtered and sorted ticket data by category and resolution time
- Built a pivot table to analyze ticket frequency and average 
  resolution time by category

## Key Finding
Account Lockout issues represent the largest category, accounting 
for 13 of 35 tickets (37%), with an average resolution time of 
approximately 0.5 hours per ticket. The high frequency suggests a 
recurring user education gap around password policies and 
self-service reset options.

## Recommendation
Create a self-service knowledge base article and short training 
video covering password reset and account unlock procedures. This 
could reduce repeat tickets in this category and free up support 
time for issues like Software Bugs, which had the highest average 
resolution time.

## Tools Used
Microsoft Excel - filtering, sorting, pivot tables
